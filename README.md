# Cervical Cancer Screening Coverage in England

An interactive map dashboard showing cervical screening coverage across England from 2010 to 2024.

## Features
- Interactive Choropleth Map (Leaflet)
- Timeline slider for historical data
- Regional breakdown in sidebar
- Data sourced from OHID Fingertips API

## Local Development
1. `npm install`
2. `python3 data-scripts/process_data.py` (to update data)
3. `npm run dev`

## Deployment
Deployed via GitHub Pages using `npm run deploy`.
